Last Update: 05.05.2023

This file contains all additional files for learning resources in Wikiversity https://de.wikiversity.org. All wikiversity files can be downloaded from GitHub as ZIP file. Subfolder /de or /en denote the language of the learning resource. The name of the folder refers to the learning resources in Wikiversity e.g. Diffusion to https://de.wikiversity.org/wiki/Diffusion for a german learning resource in Wikiversity with the name Diffusion. Click on the folder displays the learning resources in Wikiversity for which the content was created (english/german).
